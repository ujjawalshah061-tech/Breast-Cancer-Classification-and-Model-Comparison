# Breast Cancer Classification and Model Comparison

This project applies multiple machine learning algorithms to classify breast cancer tumors as malignant or benign using the Breast Cancer Wisconsin dataset. The workflow includes exploratory data analysis (EDA), cross-validation, hyperparameter tuning using GridSearchCV, feature importance analysis, and model comparison.

## Models Used

- Decision Tree Classifier
- Logistic Regression
- Random Forest Classifier
- K-Nearest Neighbors (KNN)

## Techniques Applied

- Exploratory Data Analysis (EDA)
- Stratified K-Fold Cross Validation
- GridSearchCV Hyperparameter Tuning
- Confusion Matrix Analysis
- Feature Importance Analysis
- Model Performance Comparison

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
